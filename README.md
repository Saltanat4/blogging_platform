# BlogFlow - Advanced NoSQL Final Project

### Team Members: Shinaray & Saltanat
**Course:** Advanced Databases (NoSQL)

## Project Overview
BlogFlow is a full-stack web application designed as a modern blogging platform. It demonstrates advanced MongoDB integration, including complex aggregations, indexing strategies, and secure user authentication. 

## Technical Stack
- **Frontend:** EJS, CSS, JavaScript
- **Backend:** Node.js, Express.js 
- **Database:** MongoDB (via Mongoose) 

## Key Features
- **User Authentication:** Secure register/login system.
- **Full CRUD:** Create, Read, Update, and Delete blog posts.
- **Advanced Aggregation:** Real-time tag analytics. 
- **Text Search:** Optimized searching using MongoDB indexes. 
- **Interactivity:** Nested comments and atomic like increments. 

## Installation & Setup
1. Clone the repository.
2. Install dependencies: `npm install`.
3. [cite_start]Configure your `.env` file with `MONGO_URL`. 
4. Start the server: `node index.js`.
